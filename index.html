1. Position Relative

- Stays in normal flow.
- Can be moved using top/left.
- Often used as reference for absolute elements.

2. Position Absolute

- Removed from normal flow.
- Positioned relative to nearest positioned parent.
- Can overlap other elements.

3. Z-Index

- Controls overlapping order.
- Higher value appears on top.
- Works only with positioned elements.
