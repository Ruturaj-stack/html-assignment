HTML Positioning Concepts

1. Position Relative

• The element stays in normal flow.
• It can be shifted using top/left.
• Mostly used as a reference for absolute elements.

2. Position Absolute

• Removed from normal flow.
• Positioned relative to the nearest positioned parent.
• Can overlap other elements.

3. Z-Index

• Controls which element appears on top.
• Works only on positioned elements.
• Higher z-index → front.
